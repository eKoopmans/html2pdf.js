html2pdf.js Version Tester

Use this utility to test behaviour in different versions of html2pdf.js, html2canvas, and jsPDF:

  1. Select desired version of html2pdf.js, and whether to load it with bundled dependencies.
  2. If using the unbundled version, select versions of html2canvas and jsPDF (e.g. by providing CDN URLs).
  3. Paste the HTML to be tested and render it in the iframe using the "Load in iframe" button
  4. Test the result
